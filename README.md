# node-idin

Node.js Library for [iDIN](https://www.idin.nl/)

## Prerequisites

1. Put a `cert.pem` file
1. Put a `.env` file with:
- ROUTING\_ENDPOINT
- MERCHANT\_ID
- MERCHANT\_SUBID
- MERCHANT\_RETURN\_URL
- PRIVATE_KEY
- PRIVATE_KEY_FINGERPRINT
- KEYNAME
- EXPIRATION_PERIOD
- REQUESTED_SERVICE
- DEFAULT_LANGUAGE
- LOA
- ID_PREFIX
